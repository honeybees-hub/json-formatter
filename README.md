# json-formatter

A simple and elegant JSON formatter and beautifier web tool. This project allows you to validate, format, and minify your JSON data instantly in a user-friendly interface. It features:

*   **Instant Formatting**: Quickly pretty-print or minify JSON.
*   **Error Highlighting**: Pinpoint syntax errors with line and column numbers.
*   **Dark Mode**: A sleek, attractive dark theme for comfortable viewing.
*   **Copy to Clipboard**: Easily copy formatted or minified JSON.
*   **Sample Data**: Load sample JSON to get started quickly.
